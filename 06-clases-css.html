Clases en CSS

A traves del uso de clases podemos crear estilos que pueden ser utilizados por varios elementos al mismo tiempo, ya sean encabezados (h1, h2, h3…) , parágrafos (p), etc.

A un elemento le puedo agregar varias clases pero un solo id. Por ahora agregaremos una sola clase a uno de los párrafos.
