Span

span(abarcar). Prácticamente es un contenedor dentro de una línea. Es posible crear estilos para un fragmento de un texto, ya sea un head, un parágrafo, etc. Para ello vamos a hacer uso de la etiqueta <span> La etiqueta span sirve precisamente para aplicar estilo a una parte determinada del texto.
